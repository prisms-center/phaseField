12 // Mathematical formulation (3D): S -> stress vector, R -> stress tensor, C -> stiffness tensor, E -> strain vector
13 //
14 // Generalized Hooke's law:
15 // [ S(0) ] [ C11 C12 C13 C14 C15 C16 ] [ E(0) ]
16 // [ S(1) ] [ C21 C22 C13 C14 C15 C16 ] [ E(1) ]
17 // [ S(2) ] [ C31 C32 C33 C34 C35 C36 ] [ E(2) ]
18 // [ S(3) ] = [ C41 C42 C43 C44 C45 C46 ] [ E(3) ]
19 // [ S(4) ] [ C51 C52 C53 C54 C55 C56 ] [ E(4) ]
20 // [ S(5) ] [ C61 C62 C63 C64 C65 C66 ] [ E(5) ]
21 //
22 // Strain vector definition: (using Ricci calculus notation, where commas in subscript refer to partial derivatives)
23 // E(0) = epsilon1 = epsilon11 = 1/2 ( u1,1 + u1,1 )
24 // E(1) = epsilon2 = epsilon22 = 1/2 ( u2,2 + u2,2 )
25 // E(2) = epsilon3 = epsilon33 = 1/2 ( u3,3 + u3,3 )
26 // E(3) = epsilon4 = 2*epsilon23 = ( u2,3 + u3,2 )
27 // E(4) = epsilon5 = 2*epsilon13 = ( u1,3 + u3,1 )
28 // E(5) = epsilon6 = 2*epsilon12 = ( u1,2 + u2,1 )
29 //
30 // Stress vector definition:
31 // S(0) = R[0][0] = sigma11
32 // S(1) = R[1][1] = sigma22
33 // S(2) = R[2][2] = sigma33
34 // S(3) = R[1][2] = R[2][1] = sigma23 = sigma32
35 // S(4) = R[0][2] = R[2][0] = sigma13 = sigma31
36 // S(5) = R[0][1] = R[1][0] = sigma12 = sigma21
