16 <B>What is PRISMS-PF?</B>
17 
18 PRISMS-PF is a powerful, massively parallel finite element code for conducting phase field and other related simulations of microstructural evolution. The phase field method is commonly used for predicting the evolution if microstructures under a wide range of conditions and material systems. PRISMS-PF provides a simple interface for solving customizable systems of partial differential equations of the type commonly found in phase field models, and has 11 pre-built application modules, including for precipitate evolution, grain growth, and spinodal decomposition.
19 
20 With PRISMS-PF, you have access to adaptive meshing and parallelization with near-ideal scaling for over a thousand processors. Moreover, the matrix-free framework from the deal.II library allows much larger than simulations than typical finite element programs â€“ PRISMS-PF has been used for simulations with over one billion degrees of freedom. PRISMS-PF also provides performance competitive with or exceeding single-purpose codes. For example, even without enabling the mesh adaptivity features in PRISMS-PF, it has been demonstrated to be 4.5x faster than an equivalent finite difference code.

28 <B>Quick Start Guide:</B>
29 
30 For detailed instructions on how to download and use PRISMS-PF, please consult the PRISMS-PF User Guide (the file prismspf_user_guide.pdf). A (very) abbreviated version of the instructions is given below.

32 <I>Installation:</I>
33 
34 1) Install deal.II (version 8.4.2 recommended)<br>
35  + Download CMake [http://www.cmake.org/download/]
36  + Add CMake to your path (e.g. $ PATH="/path/to/cmake/Contents/bin":"$PATH"), preferably in a shell configuration file
37  + Download and install Deal.II following instructions from from https://www.dealii.org/download.html
38  + If a Deal.II binary is downloaded, open it and follow the instructions in the terminal window
39  + If Deal.II is installed from the source, the MPI and p4est libraries must be installed as prerequisites.
40 <br>
41 
42 2) Clone the PRISMS-PF GitHub repo https://github.com/prisms-center/phaseField<br>
43  + $ git clone https://github.com/prisms-center/phaseField.git <br>
44  + $ cd phaseField <br>
45  + $ git checkout master <br>

56 <I>Running a Pre-Built Application:</I>
57 
58  Entering the following commands will run one of the pre-built example applications (the Cahn-Hilliard spinodal decomposition application in this case):<br>
59  + $ cd applications/cahnHilliard <br>
60  + $ cmake CMakeLists.txt <br><br>
61  For debug mode [default mode, very slow]: <br>
62  + $ make debug <br><br>
63  For optimized mode:<br>
64  + $ make release <br><br>
65  Execution (serial runs): <br>
66  + $ ./main <br><br>
67  Execution (parallel runs): <br>
68  + $ mpirun -np nprocs main <br>
69  [here nprocs denotes the number of processors]

79 <I>Getting started:</I>

81  Examples of various phase field models are located under the

83  run the example applications in this folder.
