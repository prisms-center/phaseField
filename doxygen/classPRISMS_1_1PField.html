Detailed Description

template<class Coordinate, class FieldType, int DIM>
class PRISMS::PField< Coordinate, FieldType, DIM >

A class for a field

Varcontainer contains variables, for instance 'std::vector<double>' for x, y, z FieldType is the datatype for the field, for instance 'double' for temperature or 'std::vector<double>' for vector displacement

A field consists of a pointer to a mesh, a list field values (at mesh nodes)
