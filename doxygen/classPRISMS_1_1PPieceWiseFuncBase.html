Detailed Description

template<class VarContainer, class OutType>
class PRISMS::PPieceWiseFuncBase< VarContainer, OutType >

Class to define a PieceWise Function

Contains a vector of 'Piece'. Throws a domain_error if it is evaluated outside of the valid domain of any piece.
