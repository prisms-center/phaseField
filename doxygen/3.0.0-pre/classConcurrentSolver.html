Solve the explicit equations.

This is a common function for solving explicit (RHS only) equations that are independent of one another.

Rather than duplicate this code a bunch of times for explicit, postprocess, amr, nucleation, etc... fields we have it here. Importantly, some of these have different functions, so we require this function.
