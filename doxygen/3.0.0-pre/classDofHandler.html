template<unsigned int dim>
std::map<unsigned int, std::unique_ptr<dealii::DoFHandler<dim> > > DofHandler< dim >::dof_handlers
private

Collection of the triangulation DoFs. The number of DoFHandlers should be equal to or less than the number of fields. Technically, there's a small optimization we can use when multiple fields have the same constraints and quadrature rule, allowing us to share the same DoFHandler. An example of this might be grain growth.
