If you have any questions, comments, or suggestions regarding PRISMS-PF, please feel free to reach out to us. We are always looking for ways to improve the code and its documentation, and your feedback is invaluable.

The development team can be reached at:

  • Email: prism.nosp@m.s-pf.nosp@m.@umic.nosp@m.h.ed.nosp@m.u
  • GitHub Discussions: PRISMS-PF Discussions
  • Mailing List: PRISMS-PF Users
Note
The development team is always happy to help, but please try and provide as much information as possible when asking for help. This includes:
  • A clear description of the problem you are facing
  • A minimal example that reproduces the issue, if possible
  • Any relevant code snippets or configuration files
  • Steps to reproduce the issue, if applicable
  • Any error messages or logs that you have encountered
  • The version of PRISMS-PF you are using
  • Your operating system and environment details
  • Any other relevant information that might help us understand the issue
