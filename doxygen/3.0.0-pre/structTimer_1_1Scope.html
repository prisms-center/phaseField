Detailed Description

Timer scope guard.

This allows you to use the timer like the following

void f()
{
Timer::Scope outer("outer");
{
Timer::Scope inner("inner");
// Work goes here
} // Inner scope ends here
// Work goes here
} // Outer scope ends here
