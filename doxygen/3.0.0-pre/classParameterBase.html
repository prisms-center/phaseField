Detailed Description

A base class for the various subcontainers of the user inputs.

There are five important steps that all child classes must achieve.

  1. Declare entries in the dealii::ParameterHandler. We can preprocess the data so users don't give us nonsensical values.
  2. Read entries from the dealii:ParameterHandler into the class.
  3. Postprocess the data into something more useful for PRISMS-PF since we know what fields exist.
  4. Validate the parameters to make sure everything is correct.
  5. Package the parameters into a minimal struct so this class can be deconstructed.
