Detailed Description

template<unsigned int dim, unsigned int degree, typename number>
class CustomPDE< dim, degree, number >

This is a derived class of MatrixFreeOperator where the user implements their PDEs.

Template Parameters
dimThe number of dimensions in the problem.
degreeThe polynomial degree of the shape functions.
numberDatatype to use. Either double or float.
